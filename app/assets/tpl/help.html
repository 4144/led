Main commands

Draw
Erase
ALT +
Pick existing value (in current layer)
ALT + SHIFT +
Pick existing value (in ANY layer)
SPACE +
Move the current level view
Zoom

Misc shortcuts

CTRL S
Save project
CTRL SHIFT S
Save project as...
CTRL W
Close project
CTRL Q
Quit app
TAB
Toggle compact mode
G
Show/hide the grid
L
Toggle grid-locking (only works with some layer types)
R
Toggle auto-layers rendering
A
Toggle active layer enhancement
F
Fit current level in screen
H
Show this help

Power-tips

S
On Tile layers, save current tile selection.
R
On Tile layers, toggle randomization of current tile selection.
SHIFT + (held)
On all supported layers, draw a rectangle.
SHIFT + (single click)
On all supported layers, flood-fill the current area with the selected value.
